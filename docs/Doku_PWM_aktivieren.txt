Verwendete Lib
https://github.com/NVIDIA/jetson-gpio


Installation:
	sudo pip install Jetson.GPIO
	sudo python3 setup.py install


WICHTTIG: 	Setting User Permissions ausführen
			sudo groupadd -f -r gpio
			sudo usermod -a -G gpio your_user_name
			sudo cp venv/lib/pythonNN/site-packages/Jetson/GPIO/99-gpio.rules /etc/udev/rules.d/
			sudo reboot


WICHTTIG: 	PWM Pins aktivieren über
			sudo /opt/nvidia/jetson-io/jetson-io.py
			-> Configure 40-pin expansion header
			-> Aktiviere PWM Kanäle
			-> Save and reboot to reconfigure pins
